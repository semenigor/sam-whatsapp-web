# SAM WhatsApp Web

**SAM WhatsApp Web** — Linux AppImage-обгортка для WhatsApp Web з додатковими функціями для роботи з повідомленнями, файлами та зручністю інтерфейсу.

Поточна стабільна версія: **v0.1.6**

## Основні можливості

- Окреме вікно для WhatsApp Web.
- Збереження входу після першого сканування QR-коду.
- Робота з Word / Excel / LibreOffice вкладеннями.
- Попередній перегляд офісних файлів.
- Відкриття офісних файлів через LibreOffice.
- SAM-копіювання кількох повідомлень.
- Збереження нумерованих списків при копіюванні повідомлень.
- SAM-закріплені чати понад стандартне обмеження WhatsApp.
- Вставка тексту і таблиць як plain text.
- Splitter між списком чатів і відкритим чатом.
- Масштабування / компактність інтерфейсу в налаштуваннях.
- Покращена шапка групових чатів.
- Довідка для користувача.
- Tray-іконка.
- Автооновлення через GitHub Releases.
- Відображення номера версії в заголовку вікна.

## Завантаження

Останню версію потрібно завантажувати зі сторінки Releases:

https://github.com/semenigor/sam-whatsapp-web/releases

Файл має вигляд:

    SAM-WhatsApp-Web-0.1.6-x86_64.AppImage

## Запуск AppImage

Після завантаження відкрийте термінал у папці з файлом і виконайте:

    chmod +x SAM-WhatsApp-Web-0.1.6-x86_64.AppImage
    ./SAM-WhatsApp-Web-0.1.6-x86_64.AppImage

Або зробіть файл виконуваним через властивості файлу в графічному інтерфейсі Linux.

## Перший вхід у WhatsApp

1. Запустіть програму.
2. Відкрийте WhatsApp на телефоні.
3. Перейдіть до підключених пристроїв.
4. Відскануйте QR-код у вікні програми.
5. Після входу сеанс має зберігатися між перезапусками.

## Оновлення

Починаючи з версії **v0.1.1**, програма підтримує перевірку оновлень через GitHub Releases.

Коли доступна нова версія, програма показує повідомлення про оновлення. Після завантаження оновлення потрібно підтвердити перезапуск.

Якщо автооновлення не спрацювало, можна вручну завантажити новий AppImage зі сторінки Releases.

## SAM-копіювання кількох повідомлень

У меню чату доступні SAM-пункти для копіювання кількох повідомлень.

Підтримується:

- вибір кількох повідомлень;
- копіювання plain text;
- збереження нумерованих списків;
- правильний розрив між окремими повідомленнями;
- вставка в LibreOffice Writer.

## SAM-закріплені чати

Програма має власний локальний механізм закріплених чатів. Це не змінює офіційні закріплення WhatsApp, а працює тільки всередині SAM WhatsApp Web.

## Вкладення Word / Excel / LibreOffice

Офісні файли можна:

- завантажувати;
- відкривати через LibreOffice;
- переглядати у preview-вікні після конвертації в PDF.

## Налаштування

У програмі є вікно налаштувань, де можна керувати:

- відкриттям офісних файлів;
- preview офісних файлів;
- кешем вкладень;
- масштабуванням / компактністю інтерфейсу.

## Системні вимоги

- Linux x86_64.
- Рекомендовано Linux Mint або сумісний дистрибутив.
- Для офісних файлів потрібен LibreOffice.

## Примітка

Це неофіційна оболонка для WhatsApp Web. Вона не є продуктом WhatsApp або Meta.
